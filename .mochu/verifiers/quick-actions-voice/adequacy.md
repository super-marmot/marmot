# Adequacy audit

Lazy artifacts that must fail:

1. A decorative microphone: the verifier requires native permission requests,
   speech result subscription, interim preview, and a pure transcript helper.
2. A voice flow that overwrites typed text: the verifier requires an append
   helper and focused tests for empty, typed, and spoken input.
3. A long unstructured chip wall: the verifier requires intent groups plus
   representative understand, write, plan, and protect actions.
4. An emoji or generic voice affordance: the verifier requires the shared
   semantic microphone icon and accessible start/stop labels.
