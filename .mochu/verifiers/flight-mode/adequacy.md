# Adequacy audit

Lazy artifacts that must fail:

1. A drawer link with no usable surface: the verifier requires stack routing,
   a real screen, an activity catalog, and focused tests.
2. An offline claim backed by the normal agent path: the verifier requires
   `allowWeb: false` and direct local engine generation.
3. A battery-risky chat clone: the verifier requires 128-token output, a 2048
   context cap, disabled thinking, and a visible stop path.
4. A fake always-on pet promise: the rubric requires an explicit no-background
   contract and leaves persistent milestones to a consented follow-up.
