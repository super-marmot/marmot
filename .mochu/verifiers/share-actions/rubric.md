# Frozen share-action-card rubric

Score the shared-text flow from 0 to 2:

- A generated transform is presented as a readable, typed result card rather
  than an unlabelled text blob.
- Draft replies clearly say they are not sent and expose only safe local
  actions such as copy or use-as-input.
- Saving shared text is previewed first and requires an explicit approval tap
  before the local document write occurs.
- Card states are understandable: preview, approved, discarded, and loading or
  failure never look like a successful write.
- The card remains readable with Markdown output and uses the same professional
  icon/button language as the chat surface.

Pass requires 8/10 or better and no criterion scored 0.
