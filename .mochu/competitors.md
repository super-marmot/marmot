# Competitive reference set

Recon is intentionally limited to product patterns for this iteration; no
competitor behavior is treated as an implementation instruction.

| Product | Reference | Relevant bar |
| --- | --- | --- |
| ChatGPT mobile | https://chatgpt.com | Composer hierarchy, clear voice/send states, restrained iconography |
| Claude mobile | https://claude.ai | Calm typography, readable message surfaces, low-noise controls |
| Apple Intelligence | https://www.apple.com/apple-intelligence/ | Native-feeling actions, clear approval boundaries, platform conventions |

Last recon: 2026-07-21. This is a qualitative UI reference set, not a claim of
feature parity.
